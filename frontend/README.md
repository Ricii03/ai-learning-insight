## Instalasi & Eksekusi Proyek

### 1. Persiapan
Pastikan **Node.js** sudah terpasang pada perangkat Anda.

### 2. Instalasi Dependensi

npm install


### 3. Menjalankan Aplikasi (Development Mode)

npm run dev

Aplikasi akan berjalan pada: http://localhost:5173/

### 4. Build untuk Produksi
npm run build
